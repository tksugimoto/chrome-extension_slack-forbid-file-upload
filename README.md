# Slack用へのファイルアップロードを禁止するChrome拡張
※ Webブラウザ版用

## ファイルアップロード禁止
- デフォルトですべてのSlackで禁止
    - インストールで自動有効化
- Slackごとに許可設定も可能
    1. 許可したいSlackページに移動
    1. 拡張のアイコンをクリック
    1. ポップアップ中の `このSlackへのファイルアップロードを許可する` チェックボックスにチェックを入れる
